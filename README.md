# MyFirstSecurity資安體驗營
```
本課程引領學生透過CTF解題來學習資訊安全的觀念與技術。
課程內容包括基礎的編碼與解碼(ASCII/BASE 64/Base32/Morse)與Linux基本觀念及入門指令。
課程將以基礎CTF解題做為提高學生學習的學習配套。
```

# 體驗內容
```
學習資安技能的超趣地圖--CTF搶旗大賽與解題實戰

登入CTF 

起手式---word隱寫術

Hitcon CTF
簡單的網頁解題

編碼與解碼

古典密碼之破密分析

linux 基礎指令與簡易安全觀念
```
